.dotfiles
=========

The *nix dot files I use on my machines

```
curl -sS https://raw.githubusercontent.com/lightster/.dotfiles/master/osx/bin/init-mac.sh >/tmp/init.mac.sh && bash /tmp/init.mac.sh && rm -f /tmp/init.mac.sh
```
